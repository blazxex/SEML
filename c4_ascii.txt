  Component Details:
  +-----------------------+----------------------------------------------+
  | Component             | Responsibility                               |
  +-----------------------+----------------------------------------------+
  | PreprocessingPipeline | Load CSVs, clean tweets, time-based 60/20/20|
  |                       | split per ticker. Output: parquet files.     |
  +-----------------------+----------------------------------------------+
  | VADERModel            | Rule-based lexicon. compound→Buy/Hold/Sell.  |
  |                       | batch=1000, ~5s, $0 cost.                    |
  +-----------------------+----------------------------------------------+
  | FinBERTModel          | ProsusAI/finbert transformer. softmax argmax.|
  |                       | conf<0.6 → No Opinion. batch=32, CUDA/MPS.  |
  +-----------------------+----------------------------------------------+
  | FinBERTFineTuner      | Fine-tunes on silver (VADER) + gold labels   |
  |                       | (200 human ×5). Early stop patience=2.       |
  |   |                   |                                              |
  |   +-> _TweetDataset   | PyTorch Dataset wrapper for tweet encodings. |
  +-----------------------+----------------------------------------------+
  | GPTModel              | 8-shot prompt → Ollama/OpenAI. batch=5.      |
  |                       | Retry ×3 backoff. Checkpoint-resumes.        |
  +-----------------------+----------------------------------------------+
  | AggregationEngine     | Group by (ticker, date). Compute label dist, |
  |                       | sentiment_score, 3d/7d rolling. Merge OHLCV. |
  +-----------------------+----------------------------------------------+
  | DriftDetector         | 4 signals: class shift>15pp, vol spike 3σ,   |
  |                       | no_opinion>35%, price divergence 5+ days.    |
  +-----------------------+----------------------------------------------+
  | EvaluationEngine      | F1/accuracy/kappa on 200 gold labels.        |
  |                       | Pearson r lags 0-3, bootstrap CI n=1000.     |
  |                       | compare_models() picks best by priority.     |
  +-----------------------+----------------------------------------------+
  | MLflow Integration    | Logs params/metrics/artifacts in child runs. |
  |                       | Registers + promotes champion to Production. |
  +-----------------------+----------------------------------------------+

  Data flow through components:

  tweets.csv ─────────────────────────────────────────────────────────────┐
  stock_prices.csv ──────────────────────────────────────────────────────┐|
                                                                         ||
  [PreprocessingPipeline]                                                ||
       reads: tweets.csv, stock_prices.csv  <────────────────────────── +|
       writes: tweets_train/val/test.parquet, prices_clean.parquet  <── +
            |
            v
  [VADERModel]       reads: tweets_test.parquet
                     writes: vader_results.csv
  [FinBERTModel]     reads: tweets_test.parquet
                     writes: finbert_results.csv
  [FinBERTFineTuner] reads: tweets_train/val.parquet + human_labels
                     writes: models/finbert_finetuned/best/
       |
       +-> [FinBERTModel (finetuned)]
                     writes: finbert_finetuned_results.csv
  [GPTModel]         reads: tweets_test.parquet
                     writes: gpt_results.csv + checkpoint.csv
            |
            v
  [AggregationEngine]
       reads: *_results.csv, prices_clean.parquet
       writes: aggregated_{model}.csv
       computes: buy_pct, sell_pct, hold_pct, no_opinion_pct,
                 sentiment_score=(buy-sell)/classified,
                 rolling_3day_sentiment, rolling_7day_sentiment
            |
            v
  [DriftDetector]
       reads: aggregated_{model}.csv
       writes: drift_flags_{model}.csv
            |
            v
  [EvaluationEngine]
       reads: *_results.csv, human_labels/lebeled.csv
       writes: classification_report_{model}.txt
       selects: best model by priority rules
            |
            v
  [MLflow Integration]
       registers: stock_sentiment_classifier
       promotes:  champion → Production

  ├──────────────────────────────────┤    ├──────────────────────────────────┤
  │ + processed_dir: str             │    │ + model_name_or_path: str         │
  │ + batch_size: int = 1000         │    │ + batch_size: int = 32            │
  ├──────────────────────────────────┤    │ + confidence_threshold: float=0.6 │
  │ + run(parent_run_id) -> DataFrame│    │ + model_key: str                  │
  │ - _classify_batch(texts) -> list │    │ + device: str  (CUDA>MPS>CPU)     │
  │ - _map_label(compound,           │    ├──────────────────────────────────┤
  │     token_count) -> tuple        │    │ + run(parent_run_id) -> DataFrame │
  ├──────────────────────────────────┤    │ - _classify_batch(texts) -> list  │
  │ Label mapping:                   │    │ - _load_model() -> void           │
  │  compound >= 0.05  → Buy         │    ├──────────────────────────────────┤
  │  compound <= -0.05 → Sell        │    │ Label mapping:                   │
  │  |compound| < 0.05 → Hold        │    │  positive → Buy                  │
  │  token_count < 3   → No Opinion  │    │  negative → Sell                 │
  +──────────────────────────────────+    │  neutral  → Hold                 │
                                          │  max_conf < 0.6 → No Opinion     │
                                          +──────────────────────────────────+

  +──────────────────────────────────+    +──────────────────────────────────+
  │        FinBERTFineTuner          │    │         _TweetDataset             │
  ├──────────────────────────────────┤    │         (PyTorch Dataset)         │
  │ + processed_dir: str             │    ├──────────────────────────────────┤
  │ + output_dir: str                │    │ + encodings: dict                 │
  ├──────────────────────────────────┤    │ + labels: list                    │
  │ + run(epochs, batch_size) -> dict│    ├──────────────────────────────────┤
  │ - _build_training_df() ->        │    │ + __len__() -> int                │
  │     DataFrame                    │    │ + __getitem__(idx) -> dict         │
  │ - _compute_metrics(eval_pred)    │    │   returns: input_ids,             │
  │     -> dict                      │    │            attention_mask,        │
  ├──────────────────────────────────┤    │            token_type_ids,        │
  │ Training data:                   │    │            labels                 │
  │  Silver: VADER on train split    │    +──────────────────────────────────+
  │          (~28k tweets)           │              created by
  │  Gold:   human_labels × 5        │    FinBERTFineTuner ──creates──> _TweetDataset
  │          (200 tweets oversampled)│    FinBERTFineTuner ──instantiates──> FinBERTModel
  +──────────────────────────────────+

  +──────────────────────────────────+
  │            GPTModel              │
  ├──────────────────────────────────┤
  │ + batch_size: int = 5            │
  │ + temperature: float = 0.0       │
  │ + max_retries: int = 3           │
  │ + ollama_model: str              │
  │ + ollama_base_url: str           │
  ├──────────────────────────────────┤
  │ + run(parent_run_id) -> DataFrame│
  │ - _call_api(tweets) -> tuple     │
  │ - _parse_response(text,n) -> list│
  │ - _build_user_message(tweets)    │
  │     -> str                       │
  ├──────────────────────────────────┤
  │ Retry: exponential backoff 2^n   │
  │ Checkpoint: saves per batch      │
  │ Fallback: "No Opinion" on fail   │
  +──────────────────────────────────+

  +──────────────────────────────────+    +──────────────────────────────────+
  │        AggregationEngine         │    │          DriftDetector            │
  ├──────────────────────────────────┤    ├──────────────────────────────────┤
  │ + processed_dir: str             │    │ + run(aggregated_df,              │
  ├──────────────────────────────────┤    │       model_name) -> DataFrame    │
  │ + run(results_df,                │    │ - _drift_flag(df) -> Series       │
  │       model_name) -> DataFrame   │    │ - _volume_spike_flag(df) -> Series│
  │ - _compute_label_distribution()  │    │ - _weak_signal_flag(df) -> Series │
  │ - _compute_sentiment_score()     │    │ - _divergence_flag(df) -> Series  │
  │ - _merge_prices(agg_df)          │    ├──────────────────────────────────┤
  ├──────────────────────────────────┤    │ Thresholds:                      │
  │ sentiment_score =                │    │  drift_flag:    class shift > 15pp│
  │   (buy - sell) / classified      │    │  volume_spike:  vol > mean + 3σ  │
  │ Range: [-1.0, +1.0]              │    │  weak_signal:   no_opinion > 35% │
  +──────────────────────────────────+    │  divergence:    5+ consec. days  │
                                          +──────────────────────────────────+

  +──────────────────────────────────────────────────────────────────────────+
  │                          EvaluationEngine                                │
  ├──────────────────────────────────────────────────────────────────────────┤
  │ + evaluate_on_human_labels(results_df, labels_csv, model_name) -> dict   │
  │     metrics: accuracy, f1_weighted, f1_macro, precision, recall, kappa   │
  │                                                                          │
  │ + evaluate_sentiment_price_correlation(agg_df, model_name,               │
  │                                        n_bootstrap=1000) -> dict         │
  │     metrics: pearson_r, p_value, ci_lower, ci_upper  @ lags 0-3         │
  │                                                                          │
  │ + compare_models(metrics: dict) -> str   (best model name)               │
  │     Priority:                                                            │
  │       1. Weighted F1 > 0.50            (gate — eliminates weak models)   │
  │       2. Highest |Pearson r|           (sentiment-price alignment)       │
  │       3. Lowest no_opinion_rate        (more actionable signal)          │
  │       4. Highest inter-model agreement (reliability across models)       │
  │       5. Lowest cost                   (VADER < FinBERT < GPT)          │
  │                                                                          │
  │ + inter_model_agreement(*result_dfs, model_names) -> DataFrame           │
  │     computes pairwise agreement rate for all model pairs                 │
  │ - _bootstrap_pearson(x, y, n) -> (ci_lower, ci_upper)                   │
  +──────────────────────────────────────────────────────────────────────────+

  +──────────────────────────────────────────────────────────────────────────+
  │                          Airflow DAG                                     │
  │              dag_id: sentiment_retrain_pipeline                          │
  │              schedule: "0 2 * * *"  (daily 02:00 UTC)                   │
  ├──────────────────────────────────────────────────────────────────────────┤
  │                                                                          │

  │  [retrain_finbert]      PythonOperator                                   │

  │       v                                                                  │
  │  [evaluate_challenger]  PythonOperator                                   │

  │       XCom push: challenger_f1, challenger_metrics                      │
  │       |                                                                  │
  │       v                                                                  │
  │  [compare_champion]     PythonOperator                                   │
  │       loads champion F1 from MLflow Production stage                    │
  │       pulls challenger_f1 from XCom                                     │

  │       |                                                                  │
  │       v                                                                  │
